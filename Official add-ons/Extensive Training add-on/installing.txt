== NOTICE ==
This is an official add-on, it is made by the map creator and will be updated alongside the map.
Any compatibility issues with this add-on should be reported to the conflicting add-on.

The 11 ghosts by yourself in the training room advancement is not granted for this add-on's training room programs. Select vanilla to try to get that instead.
The capturing of ghosts in the training room tasks from the 3DS Remake add-on will not be granted for this add-on's training room programs. Select 3DS Remake to try to get that instead.

== How to install ==
1. Open the Luigi's Mansion world folder.
2. Open the "datapacks" folder
3. Drop the data pack folder (named "Extensive Training") in this folder

== Updates ==
When updating the map, delete the existing Extensive Training add-on and re-install the latest version.

== Latest compatible world version ==
Same version as this folder is shipped with.
